# Git Push

Nu je de opdrachten allemaal op je eigen GitHub account hebt, een branche voor je opdracht, en je weet hoe je dit op VSCode krijgt, zijn jullie klaar opdrachten te maken.

Als je lokaal (dus op VSCode) je opdracht hebt gemaakt met de juiste commits, gaan we pushen naar GitHub.
Als het goed is, is dit al bekend, zo ja: ga dan verder naar de pull requests.
Zo niet, dan staat het hier ook onder bij het voorbeeld.
